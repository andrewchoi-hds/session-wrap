# session-wrap

Claude Code 세션 마무리 자동화 플러그인

## 이런 경험 있으신가요?

세션 끝나면 뭔가 찝찝합니다:

- "이거 CLAUDE.md에 적어야 하나?"
- "다음에 뭐 해야 하더라?"
- "오늘 배운 거 정리해야 하는데..."
- "이 패턴 자동화하면 좋을텐데..."

**`/wrap` 하나면 해결됩니다.**

## 왜 session-wrap인가?

### 문제

| 기존 방식 | 문제점 |
|----------|--------|
| 수동 정리 | 까먹음, 귀찮음, 일관성 없음 |
| 커밋만 하고 끝 | 배운 것, 다음 할 일 사라짐 |
| CLAUDE.md 직접 수정 | 뭘 적어야 할지 모름 |

### 해결

| session-wrap | 효과 |
|--------------|------|
| 5개 AI 에이전트가 세션 분석 | 놓치는 것 없음 |
| 문서 업데이트 자동 제안 | CLAUDE.md가 풍부해짐 |
| 자동화 기회 탐지 | 반복 작업 감소 |
| TIL 자동 추출 | 배운 것이 쌓임 |
| 다음 할 일 정리 | 다음 세션 시작이 빠름 |

## 설치

### 방법 1: 원라인 설치 (추천)
```bash
curl -sL https://raw.githubusercontent.com/andrewchoi-hds/session-wrap/main/install.sh | bash
```

업데이트할 때도 같은 명령어를 실행하면 됩니다.

### 방법 2: zip 파일
```bash
# 압축 해제 후 .claude 폴더를 프로젝트로 복사
unzip session-wrap-plugin.zip
cp -r .claude /your/project/
```

### 방법 3: 직접 복사
```bash
cp -r session-wrap/.claude /your/project/
```

## 사용법

```bash
/wrap
```

또는 자연어로:

```
세션 정리해줘
마무리하자
오늘은 여기까지
```

**25개 이상의 한국어 트리거를 지원합니다.**

## 지원하는 트리거

### 명령어
- `/wrap`

### 한국어 (자연어)
| 유형 | 트리거 |
|------|--------|
| 기본 | 세션 정리 / 세션 마무리 / 세션 끝 |
| 요청 | 마무리해줘 / 정리해줘 |
| 대화 | 이제 정리하자 / 오늘은 여기까지 |
| 간단 | 끝내자 / 마치자 |
| 종료 | 오늘 끝 / 오늘 작업 끝 |
| 질문 | 커밋할거 있어 / 뭐 기록해야해 |
| 축약 | 랩 / 랩업 |

### 영어
- `wrap up session` / `end session` / `session wrap`

## 동작 방식

### Phase 1: 병렬 분석 (4개 에이전트)

| 에이전트 | 하는 일 |
|----------|---------|
| `doc-updater` | CLAUDE.md에 추가할 내용 제안 |
| `automation-scout` | 반복 패턴 → skill/command 자동화 제안 |
| `learning-extractor` | 배운 것, 실수, 발견 추출 (TIL) |
| `followup-suggester` | 미완성 작업, 다음 세션 우선순위 |

**4개가 동시에 실행되어 빠릅니다.**

### Phase 2: 중복 검증

| 에이전트 | 하는 일 |
|----------|---------|
| `duplicate-checker` | 이미 있는 내용인지 검증, 중복 제거 |

### Phase 3: 사용자 선택

분석 끝나면 선택지를 줍니다:

- [ ] 변경사항 커밋
- [ ] CLAUDE.md 업데이트
- [ ] 자동화 생성
- [ ] 배운 것 저장
- [ ] 후속 작업 저장

**원하는 것만 선택하면 됩니다.**

## 출력 예시

```
## 세션 마무리 완료

### 문서 업데이트
- CLAUDE.md에 3개 제안

### 자동화 기회
- 2개 잠재적 자동화 발견

### 배운 것
- 4개 TIL 추출

### 후속 작업
- 다음 세션에 5개 작업

무엇을 할까요?
```

## 장점 요약

| 장점 | 설명 |
|------|------|
| **시간 절약** | 세션 정리에 쓰는 시간 90% 감소 |
| **놓침 방지** | 5개 에이전트가 모든 측면 분석 |
| **지식 축적** | CLAUDE.md, TIL이 자동으로 풍부해짐 |
| **일관성** | 매 세션 동일한 품질의 정리 |
| **자연어** | 한국어로 편하게 사용 |

## 플러그인 구조

```
.claude/
├── agents/
│   ├── doc-updater.md        # 문서 업데이트 제안
│   ├── automation-scout.md   # 자동화 기회 탐지
│   ├── learning-extractor.md # 배운 것 추출
│   ├── followup-suggester.md # 다음 할 일 제안
│   └── duplicate-checker.md  # 중복 검증
├── commands/
│   └── wrap.md               # /wrap 스킬 정의
└── settings.json             # 설정
```

## FAQ

### Q: 다른 프로젝트에서도 쓸 수 있나요?
A: 네, `.claude/` 폴더만 복사하면 어느 프로젝트에서든 작동합니다.

### Q: 영어로 출력하고 싶어요
A: 현재는 한국어 출력이 기본입니다. 영어 지원은 향후 업데이트 예정입니다.

### Q: 커스터마이징 가능한가요?
A: `.claude/agents/` 폴더의 마크다운 파일을 수정하면 됩니다.
